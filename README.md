# jakobferdinand.at

Here is the sourcecode of my homepage [jakobferdinand.at](https://jakobferdinand.at).

## Development

Static site. Edit `index.html` / `style.css` and push to `main`; Azure Static Web Apps deploys automatically.

## Local development

Serve the repo over HTTP with Python's built-in server:

```bash
python3 -m http.server 8000
```
